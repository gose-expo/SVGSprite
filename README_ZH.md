## 📝 项目描述

这是一个包含28个主流平台和品牌图标的 SVG Sprite 集合，专为现代 Web 开发设计。所有图标采用统一的 24×24 像素 viewBox，确保在不同尺寸下都能保持清晰的显示效果。

## ✨ 特性

* **🎯 全覆盖** - 涵盖社交媒体、通讯工具、开发平台、娱乐服务等28个热门平台
* **🚀 高性能** - 单文件加载，减少 HTTP 请求，提升页面加载速度
* **🎨 易定制** - 通过 CSS 轻松控制颜色、大小和样式
* **📱 响应式** - 矢量图标支持任意缩放，适配各种设备和分辨率
* **🔧 易使用** - 统一的命名规范和简洁的调用方式
* **💾 可缓存** - 浏览器缓存友好，提升用户体验

## 🚀 快速开始

### 方法1：内联使用（推荐）

html

```html
<!-- 1. 在页面中嵌入 SVG Sprite -->
<svg xmlns="http://www.w3.org/2000/svg" style="display: none;">
  <!-- 完整的 SVG Sprite 内容 -->
</svg>

<!-- 2. 使用图标 -->
<svg width="24" height="24">
  <use href="#icon-instagram"></use>
</svg>
```

### 方法2：外部文件

html

```html
<!-- 1. 保存为 icons.svg 文件 -->
<!-- 2. 引用图标 -->
<svg width="24" height="24">
  <use href="icons.svg#icon-instagram"></use>
</svg>
```

## 🎨 样式定制

### 基础样式

css

```css
/* 尺寸控制 */
.icon-small { width: 16px; height: 16px; }
.icon-medium { width: 24px; height: 24px; }
.icon-large { width: 32px; height: 32px; }

/* 颜色控制 */
.icon-blue { fill: #1877F2; }
.icon-red { fill: #E4405F; }
```

### 品牌色彩

css

```css
.instagram { fill: #E4405F; }
.facebook { fill: #1877F2; }
.twitter { fill: #1DA1F2; }
.youtube { fill: #FF0000; }
.github { fill: #181717; }
.spotify { fill: #1DB954; }
```

### 在按钮中使用

html

```html
<button class="social-btn">
  <svg class="icon-medium instagram">
    <use href="#icon-instagram"></use>
  </svg>
  关注我们
</button>
```
